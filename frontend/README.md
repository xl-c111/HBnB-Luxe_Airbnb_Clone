# HBnB Luxe — Frontend

React + TypeScript single‑page app for the HBnB Luxe project.

For the full project overview (architecture, backend, deployment), see `../README.md`.

## Local Development
```bash
cp .env.example .env
npm install
npm run dev
```

## Environment Variables
- `VITE_API_URL` (default: `http://localhost:5000`)
- `VITE_STRIPE_PUBLISHABLE_KEY` (Stripe test publishable key)

## Tests
```bash
npm test
```
